## 💻 Tecnologias: 
* HTML;
* CSS;
* JavaScript.

## 📝 Objetivos:
- [x] Responsividade;
- [x] Tags semânticas;
- [x] FlexBox e Grid;
- [x] Animação CSS;
- [x] JavaScript para interação com usuário - DOM; 
- [ ] Dark Mode; 
- [ ] Visualização de senha; 
- [ ] Validação de formulário;
